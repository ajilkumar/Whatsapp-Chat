
## WhatsApp Chat without Save
## Installation

### Clone the Repository

```bash
git clone (https://github.com/ajilkumar/Whatsapp-Chat.git)
cd WhatsappChat
```
### Install dependencies
```console 
npm install
```
### Start the client
```console
npm start
```
```python
The Website will be available at http://localhost:3000
```
